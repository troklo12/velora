# VELORA — Premium Fashion E-Commerce

A fully responsive, premium fashion boutique website built with HTML, CSS, JavaScript, and Bootstrap.

## Project Structure

```
velora/
├── index.html        # Home page
├── catalog.html      # Product catalogue / listing page
├── product.html      # Product detail page
├── stylist.html      # Personal stylist form page
├── style.css         # Custom VELORA brand stylesheet
├── script.js         # JavaScript interactions & form validation
├── favicon.svg       # Brand favicon
└── README.md
```

## Pages

| Page | Description |
|---|---|
| `index.html` | Hero, categories, editorial grid, new arrivals, trending looks, bestsellers, seasonal banner, newsletter |
| `catalog.html` | Product grid with real-time search, category/size filters, sort dropdown, pagination |
| `product.html` | Product gallery, size/colour selectors, add to cart, accordions, complete the look, reviews |
| `stylist.html` | Personal stylist request form with full validation, success/error modals |

## Features

- ✅ Fully responsive (mobile, tablet, desktop)
- ✅ Real-time product search filter
- ✅ Form validation (email regex, phone regex) with Bootstrap Modals
- ✅ Scroll-triggered fade-up animations
- ✅ Sticky navbar with scroll state
- ✅ Animated ticker strip
- ✅ Wishlist toggle
- ✅ Size & colour selectors
- ✅ Add to cart with cart badge update
- ✅ Filter sidebar (category, size, price range)

## Deployment (GitHub Pages)

1. Push this repository to GitHub
2. Go to **Settings → Pages**
3. Source: **Deploy from a branch**
4. Branch: `main` / `root`
5. Save — your site will be live at `https://[username].github.io/velora/`

## Tech Stack

- HTML5 + CSS3 (CSS Variables, Grid, Flexbox)
- Bootstrap 5.3
- Bootstrap Icons 1.11
- Vanilla JavaScript (ES6+)
- Google Fonts: Cormorant Garamond, Playfair Display, Montserrat
